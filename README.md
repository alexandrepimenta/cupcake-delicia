# Cupcake Delícia (versão de demonstração - SQLite)
Projeto para o PIT II - Aplicativo de venda e personalização de cupcakes.

## Como usar (passo a passo)
1. Instale Node.js (versão 14+).
2. No terminal, dentro da pasta do projeto, rode:
   npm install
3. Inicie o servidor:
   npm start
4. Abra no navegador:
   http://localhost:3000

## Observações
- O projeto usa **SQLite** (arquivo em `data/cupcake.db`), portanto não é necessário instalar um servidor de banco.
- Se for necessário migrar para MySQL mais tarde, eu te ajudarei a ajustar as configurações.
